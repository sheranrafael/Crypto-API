# Crypto API 

A simple API for real-time cryptocurrency price queries, developed with Ruby on Rails.

## Features

- Bitcoin and Ethereum price lookup
- Currency conversion
- API health check
- Integrated frontend with HTML/JS/CSS

## Technologies

- Ruby 3.3.7
- Rails 8.0.2
- SQLite 3.49.1
- HTTParty (integration with APIs)
- CoinGecko API
